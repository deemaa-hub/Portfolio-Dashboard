# 📊 Responsive Portfolio Dashboard with Admin Access

A dynamic and professional dashboard designed specifically to manage and showcase software development projects. This project was built using clean code practices, separating structure, styling, and business logic while ensuring full responsiveness and accessibility.

---

## ✨ Key Features

* **Pre-loaded Projects:** Automatically displays 5 core projects (including the interactive Calculator App) for any first-time visitor.
* **Admin Authentication:** Features a secure login mechanism allowing only authorized administrators to add or delete projects.
* **Local Storage Persistence:** Syncs all modifications, additions, and deletions with the browser's Local Storage, ensuring data remains intact upon page refresh.
* **Fully Responsive UI:** Modern and sleek design that adapts flawlessly to all screen sizes (Desktops, Tablets, and Mobile phones) using advanced CSS Media Queries.
* **Secure Coding Practices:** Implements proper DOM manipulation (via `textContent`) to safeguard the application against potential XSS (Cross-Site Scripting) vulnerabilities.

---

## 🛠️ Tech Stack

* **HTML5:** Built with semantic tags (`<aside>`, `<main>`, `<section>`) to ensure maximum SEO efficiency and accessibility.
* **CSS3:** Advanced styling using CSS Variables for consistent color branding and flexible layout management.
* **JavaScript (ES6+):** Pure Vanilla JS to handle application logic, state management, security checks, and the Local Storage API.
